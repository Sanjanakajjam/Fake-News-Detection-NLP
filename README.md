# Fake News Detection using NLP

## 📌 Overview
This project detects whether a news article is fake or real using Natural Language Processing (NLP) and Machine Learning techniques.

## 🚀 Features
- Text preprocessing using NLTK
- TF-IDF feature extraction
- Logistic Regression classifier
- Streamlit web interface
- Real/Fake news prediction

## 🛠️ Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Streamlit

## 📊 Dataset
Dataset taken from Kaggle:
Fake and Real News Dataset

## ⚙️ Workflow
1. Load Dataset
2. Text Cleaning
3. TF-IDF Vectorization
4. Model Training
5. Prediction

## 🎯 Result
The model achieved high accuracy in detecting fake and real news articles.

## ▶️ How to Run
1. Open notebook in Google Colab
2. Upload Fake.csv and True.csv
3. Run all cells
